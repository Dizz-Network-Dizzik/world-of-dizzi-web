Was es zusammenhält

Drei Entscheidungen, alles Weitere folgt

Lokal-first

Der Core läuft offline. Cloud-Dienste sind zuschaltbare Konnektoren — jederzeit abschaltbar, immer als externer Schritt sichtbar. Das macht Synchronisation schwerer; wir haben uns bewusst dafür entschieden, weil die Daten auf die Maschine gehören, die sie erzeugt hat.

Vertrags-getrieben

Ein geschriebener App-Vertrag, zehn Apps, die ihn erfüllen. Wer ihn erfüllt, erscheint von selbst im Dashboard und ist für die KI ansprechbar. Schreib den Vertrag — die App erscheint.

Ehrlich per Gesetz

Ehrlichkeit ist hier kein Tonfall, sondern eine geschriebene Regel, an die sich der Code halten muss: ein nicht gemessener Wert wird als „nicht gemessen" ausgegeben, nie als Null. Dieselbe Regel gilt für diese Website.

Die Methode

Gesetze vor Code

Ein System dieser Größe ist für eine Person nicht baubar ohne Regeln, die auch dann tragen, wenn niemand hinsieht. Sie stehen geschrieben im Repository, und sie sind älter als der meiste Code: Gesetze vor Code, Verträge vor Funktionen, Recherche vor Entscheidungen.

Ein Mensch führt Regie. Ein verwaltender Chat hält den Gesamtstand und entscheidet, was als Nächstes passiert; getrennte Bau-Chats erledigen die Arbeit, ein Arbeitspaket nach dem anderen. Sie sprechen nicht miteinander, sondern über geschriebene Übergabe-Dokumente. Und alles Unumkehrbare — Geld, Veröffentlichen, Löschen — steht hinter einem menschlichen Gate.
